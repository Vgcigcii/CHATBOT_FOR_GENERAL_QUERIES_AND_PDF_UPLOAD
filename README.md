# Chatbot for general queries and pdf uploads 
This project allows user to make general queries and also upload pdfs to the chatbot for queries related to it.
## Installation
To install the required dependencies, run the following command:
```bash
pip install -r requirements.txt
pip install ollama
pip install os

## Usage
To run the chatbot, execute the following command:
```bash 
python main.py
```
This will start the chatbot and you can interact with it by typing your queries related to ESG.
